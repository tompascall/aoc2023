# aoc2023

Play with aoc

## Prerequisites

node v20

## How to run

```
EXPORT AOC_SESSION_COOKIE=<your advent of code session cookie>
```

### Run build

```
npx tsc
```

### Run build with watch mode

```
npx tsc --watch
```

### Run tests

```
npm test -- --watch
```

### Get day n solution

```
node dist/day-01/index.js
```
